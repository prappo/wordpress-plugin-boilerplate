## Install
```bash
npx wordpress-plugin-boilerplate
```
Or 
```bash
git clone https://github.com/prappo/wordpress-plugin-boilerplate.git
```
```bash
cd wordpress-plugin-boilerplate
```
```bash
composer install
```
```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Release

```bash
npm run release
```

It will create a relase plugin in `release` folder
